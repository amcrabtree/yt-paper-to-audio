
# Paper to Audio

Convert a scientific paper into an entertaining audio summary using an LLM and the Kokoro text-to-speech model.

## Requirements

- Python 3.13+
- [uv](https://docs.astral.sh/uv/getting-started/installation/)
- An Anthropic or OpenAI API key

## Setup

**1. Clone the repo and install dependencies**

```bash
git clone https://github.com/amcrabtree/yt-paper-to-audio
cd yt-paper-to-audio
uv sync
```

**2. Create a `.env` file**

```bash
cp .env.example .env
```

Then open `.env` and set one of the following:

```
ANTHROPIC_API_KEY=your_anthropic_api_key_here
# or
OPENAI_API_KEY=your_openai_api_key_here
```

You only need one. If both are set, the Anthropic key is used by default.

To use a specific model, add:

```
MODEL=anthropic/claude-sonnet-4-6
# or
MODEL=openai/gpt-4o-mini
```

If `MODEL` is not set, it defaults to `anthropic/claude-sonnet-4-6`.

**3. Add your paper**

Place the PDF you want to summarize in the `papers/` directory.

## Running the CLI

```bash
uv run python paper_to_audio.py papers/<your_paper>.pdf
```

The script runs in four steps:

1. **Convert PDF to text** — extracts the full text from the paper PDF
2. **Prepare the prompt** — combines the paper text with narration instructions
3. **Generate the summary** — sends the prompt to the LLM and saves the script to `output/`
4. **Convert to audio** — synthesizes speech sentence-by-sentence using Kokoro and exports an MP3 to `output/`

### Optional arguments

| Argument | Default | Description |
|----------|---------|-------------|
| `--model` | `anthropic/claude-sonnet-4-6` (or `$MODEL`) | LLM model to use (litellm format) |
| `--output-dir` | `output` | Base directory for output files |
| `--skill-file` | `.claude/skills/paper-to-audio-summary/SKILL.md` | Path to the narration instructions file |
| `--voice` | `af_heart` | Kokoro voice to use |
| `--speed` | `1.0` | Speech speed |

## Output

Results are saved to `output/<paper_stem>/`:

| File | Description |
|------|-------------|
| `<paper>.txt` | The raw text extracted from the PDF |
| `<paper>_summary.txt` | The narration script generated by the LLM |
| `<paper>_summary.mp3` | The final audio file |
